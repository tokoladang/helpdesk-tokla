## 1. **Input nomor PO** pada kolom pencarian dan klik **ENTER**.

![01kh36qtdghyf8ew6s5e350j82](https://cdn-pusatinformasi.siplahtokoladang.id/resources/images/01kh36qtdghyf8ew6s5e350j82.webp)

## 2. Klik tombol **Detail**

![01kh36rr23v53j91kc00whn5w5](https://cdn-pusatinformasi.siplahtokoladang.id/resources/images/01kh36rr23v53j91kc00whn5w5.webp)

## 3. Tutup dengan klik X

![01kh384dnwmtxrp4ctvk7ngsdp](https://cdn-pusatinformasi.siplahtokoladang.id/resources/images/01kh384dnwmtxrp4ctvk7ngsdp.webp)

## 4. Klik **Ajukan Penyelesaian Transaksi**

![01kh384ydx24se4mkxem2fnc95](https://cdn-pusatinformasi.siplahtokoladang.id/resources/images/01kh384ydx24se4mkxem2fnc95.webp)

## 5. Dibaca dan dipahami tentang pengajuan ini dan klik tombol **YA**

![01kh387573wxkgp6sm5enr50ch](https://cdn-pusatinformasi.siplahtokoladang.id/resources/images/01kh387573wxkgp6sm5enr50ch.webp)

## 5. Unduh dokumen **Surat Kesepakatan Penyelesaian Transaksi**

![01kh387m0x6t5hz17f6cyfv4g6](https://cdn-pusatinformasi.siplahtokoladang.id/resources/images/01kh387m0x6t5hz17f6cyfv4g6.webp)

## 6. Silahkan tanda tangan, stempel sekolah, dan bubuhkan **Meterai 10.000** pada surat tersebut.

## 7. Scan dokumen jadikan file format **PDF** dengan ukuran file maksimal 3MB.

## 8. Pilih alasan dan unggah file pada menu seperti gambar di bawah ini.

![01kh38qarpfnev2bhx1qxcnyc0](https://cdn-pusatinformasi.siplahtokoladang.id/resources/images/01kh38qarpfnev2bhx1qxcnyc0.webp)

## 9. Klik tombol **Ajukan penyelesaian**.

## 10. Selesai.
